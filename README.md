캡스톤디자인과 ICT해상물류에서 사용한 드론제어 코드입니다. 

DroneKit-Python라이브러리를 사용하여 드론의 포인트비행을 코드로 구현하였으며,

HTTP통신을 이용하여 웹서버와 통신하여 순찰드론, 라즈베리파이, 경고드론의 State를 변경 및 정보를 읽어와 드론을 제어합니다.
